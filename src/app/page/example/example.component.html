Descripción general

Esta aplicación es un sistema web desarrollado con Angular (v20), utilizando una arquitectura modular y escalable para gestionar usuarios, roles y notas. Incorpora Angular Material y TailwindCSS para lograr una interfaz moderna, limpia y responsiva.

Tecnologías utilizadas

  • Angular: Framework principal para el frontend.
  • Angular Material: Componentes UI listos para usar.
  • TailwindCSS: Utilidades CSS para estilos rápidos y personalizables.
  • RxJS: Programación reactiva para manejo de datos asíncronos.
  • JWT: Autenticación basada en tokens JSON Web Token.
  • TypeScript: Lenguaje principal para el desarrollo.
  • PostCSS: Procesador de CSS.

Soluciones implementadas

  • Autenticación y autorización: Registro, login y protección de rutas usando JWT y guards personalizados.
  • Gestión de usuarios y roles: CRUD con paginación y modales.
  • Gestión de notas: CRUD de notas personales con UI amigable.
  • Interfaz modular: Módulos claros como auth, dashboard, notas, etc.
  • Validaciones avanzadas: Formularios con validaciones personalizadas.
  • Internacionalización: Uso de signals, detección de cambios optimizada y estructura limpia.
